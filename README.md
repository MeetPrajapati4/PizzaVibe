# 🍕 PizzaVibe

> A full-stack pizza ordering platform with a modern frontend, backend services, admin features, and a live deployment.

## ✨ Overview
PizzaVibe is a **full-stack web application** built for a smooth pizza browsing and ordering experience. The repository includes separate `frontend` and `backend` folders, along with Docker support and scripts for development workflows. [1]

## 🚀 Highlights
- 🍕 Modern pizza ordering experience.
- 🧑‍💼 Admin authentication and admin features.
- 🗂️ Separate frontend and backend structure.
- 🐳 Docker Compose support for easier setup.
- 🌐 Live deployment available online. [1]

## 📦 Getting Started
### 1️⃣ Clone the repository
```bash
git clone https://github.com/MeetPrajapati4/PizzaVibe.git
cd PizzaVibe
```

### 2️⃣ Install dependencies
```bash
npm install
```

### 3️⃣ Run the project
Because the exact start commands are not shown on the repository page, use the project scripts defined in `package.json` or Docker Compose as needed. [1]

```bash
npm run dev
```

Or, if Docker is configured for your workflow:

```bash
docker compose up --build
```
